Images

Both the above photos are relative path links. That means that they refference the image relative to where the page is located in the file structure. This means if I were to move this page to a differnt folder I would have to updated the src link.

These also have some other differences. The first image has a title attribute which allows you to hover your mouse over it and get a little message box that describes what the image is. the second has a figure tag applied to it and a figcaption that allows you to group a caption with the image. the figure and figcaption can be used on anything though. not only images.

Both images have their width set to 700px

This image is an absolute link. This means that it is linked from the root folder or in this case its absolute web address. In this case if i were to move this page into a different folder the link would still work. However it is best to use relative links whenever possible becasue they take less time to load.
